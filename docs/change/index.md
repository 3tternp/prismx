# 更新日志

#### 1.1.0 （2024-1-17）：

- 修复 前端端口过多导致的卡顿问题
- 修复 前端分辨率过低导致 UI 错位问题
- 新增 Nuclei 引擎，nuclei 的 poc 请手动放置 `lib\exploits\nuclei` 目录

#### Tips:

> Nuclei 引擎默认为禁用，在创建扫描任务时在高级设定中手动开启。遇到过多 web 端口请谨慎使用，经测试扫描一个 C 段，不开 nuclei 引擎只占用系统约 300m 内存，开启后占用到了 2G 以上
